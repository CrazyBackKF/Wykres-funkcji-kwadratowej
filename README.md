# 📈 Rysowanie wykresu funkcji kwadratowej

Prosta strona internetowa umożliwiająca rysowanie wykresu **funkcji kwadratowej** na podstawie podanych współczynników.

Aplikacja pozwala użytkownikowi wprowadzić wartości **a**, **b** oraz **c** dla funkcji:

\[
f(x) = ax^2 + bx + c
\]

Po podaniu współczynników wykres funkcji jest automatycznie rysowany na układzie współrzędnych.

---

## ⚙️ Funkcjonalności

- wprowadzanie współczynników **a**, **b**, **c**
- dynamiczne rysowanie wykresu funkcji
- wyświetlanie paraboli na układzie współrzędnych
- aktualizacja wykresu po zmianie wartości

---

## 🧮 Jak działa aplikacja

1. Użytkownik podaje wartości współczynników **a**, **b** i **c**.
2. Program oblicza wartości funkcji dla kolejnych punktów **x**.
3. Na podstawie obliczeń rysowany jest wykres funkcji w postaci paraboli.

Funkcja opisana jest wzorem:

\[
f(x) = ax^2 + bx + c
\]

gdzie:
- **a** – decyduje o kierunku i "szerokości" paraboli,
- **b** – wpływa na przesunięcie wykresu,
- **c** – określa punkt przecięcia z osią **Y**.

---

## 🖥️ Technologie

Projekt został wykonany przy użyciu:

- **HTML** – struktura strony  
- **CSS** – stylizacja  
- **JavaScript** – obliczenia i rysowanie wykresu  
- **Canvas** – rysowanie wykresu na stronie
